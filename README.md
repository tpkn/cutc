## Usage

```
cutc [ -options ] < <file.csv>
```

## Options

```
-d, --delimiter    Fields delimiter
-f, --fields       Fields indexes to cut (starting from 1, order matters)
-h, --header       Skip csv header
--help             Help
--version          Version
```
